## 标题句式

- 反差/拟人型：把抽象事物拟人 + 情绪反转。例：「冰冷的同事变成温暖的skill」。
- 结果前置 + 低投入承诺型：数字结果 + 省力路径。例：「三周低强度速通GRE330」。
- 差异化否定型：用「不××版」「不海投版」制造反套路钩子。
- 清单/保姆级教程型：「保姆级教程」「超长干货」明示收藏价值，常配括号补充。
- 成就播报型：荣誉 + emoji 强化（🏅️），短促有力。
- 互动引导型：把动作写进标题，「左滑帮我入职……👈🏻」。
- 极简留白型：3–7 字一句生活感短句，「这倒是提醒我了」「ICLR结束 开始放假」，靠正文/图承接。
